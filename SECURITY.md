# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| <= 0.14 | :x:                |
| 0.15.x  | :white_check_mark: |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

For security issues, kindly email us at security@appwrite.io instead of posting a public issue in GitHub.
